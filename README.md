# upgraded-adventure - https://dwarforca.com/

This repository now includes a minimal static website for the project.


## Preview locally

You can preview the site locally using a simple static server. From the repository root run:

```
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser and view `index.html`.

Files added:

- `index.html` — main static page
- `assets/styles.css` — simple stylesheet
